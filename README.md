#  Quiz App 

### Features:
The Quiz App includes the following features:

1. Header Section:

- Displays a logo and navigation links.
- Provides a user account section with profile image and options.

2. Main Section:

- Shows the title of the quiz.
- Score board

3. Submit Button:

- Allows users to submit their quiz answers.

4. Quiz Score Popup Box:

- Displays the quiz score after submission.
- Shows the number of correct answers out of the total questions.

5. Interactive Design:

- The quiz interface is visually appealing and interactive similar to PW Skills Quiz Website.

6. My Account Hover:

- The user account information is shown or hidden on hover.

### Technologies Used
The Quiz App is built using the following technologies:

- HTML: Used for creating the structure of the web page.
- CSS: Used for styling the web page and making it visually appealing.
- JavaScript: Used for implementing the quiz functionality, handling user interactions, 
  and displaying quiz scores.
- Font Awesome: Imported to include icons in the web page for various elements.

### How to Use

- Clone or download the project repository.
- Open the HTML file in a web browser to start the quiz.
- Answer the quiz questions by selecting the appropriate options.
- Click the "Submit" button to see your quiz score.
